Como Usar o Guara-system

Bem-vindo ao Guara-system! Este webapp foi desenvolvido para gerenciar Licitações e POs de forma rápida e eficiente. Siga as instruções abaixo para tirar o máximo proveito do sistema.

1. Acesso ao Sistema e Login

  • Ao acessar o sistema, você será redirecionado para a página de login, onde deverá inserir seu email e senha.
  • Se ainda não possuir uma conta, clique em "Criar conta" para se registrar.

2. Navegação no Dashboard

  • Após o login, você será direcionado ao Dashboard, onde as licitações cadastradas são exibidas.
  • Utilize o menu superior para alternar entre Licitações (Dashboard) e Controle de POs.
  • O ícone de perfil (representado por ) no canto superior direito permite acessar as opções de Resetar Senha e Sair.

3. Gerenciamento de Licitações

  • Adicionar Nova Licitação: Clique em "Adicionar nova licitação" no menu flutuante. Preencha os campos obrigatórios, incluindo o CMM (Consumo Mensal Médio) e o Saldo (Disp. p/lib). A Autonomia é calculada automaticamente (Saldo ÷ CMM).
  • Editar/Excluir: Utilize os ícones de lápis e lixeira em cada cartão para editar ou excluir uma licitação.
  • Exportar Dados: Clique no botão "Exportar" para baixar os dados em formato Excel (.xls).
  • Importar Estoque: Use o botão "Importar Estoque" para carregar uma planilha (CSV ou Excel) com as colunas NOME_ITEM e QTDE_DISPONIVEL para atualizar o saldo.

4. Gerenciamento de POs

  • Adicionar Novo PO: Clique em "Adicionar novo PO" e preencha o formulário. Selecione a licitação relacionada.
  • Converter PO para OC: No cartão do PO, clique no botão para converter o PO em OC e insira o número da OC quando solicitado.
  • Editar/Excluir/Arquivar: Utilize os ícones correspondentes para editar, excluir ou arquivar POs.
  • Exportar Dados: Clique no botão "Exportar" para baixar os dados de POs em formato Excel (.xls).

5. Pesquisa e Filtros

Utilize as barras de pesquisa para filtrar licitações ou POs por nome. Você também pode usar os filtros por categoria (Todos, Frigorificados ou Secos) para refinar os resultados.

6. Comentários e Verificação de OCs

  • Comentários: Clique no ícone de chat () em cada licitação para visualizar e adicionar comentários.
  • Verificar OCs: Clique em "Verificar OCs" para visualizar os OCs associados à licitação, incluindo os que estão aguardando aceite e os arquivados para controle.

7. Perfil e Segurança

  • Acesse o menu de perfil clicando no ícone no canto superior direito.
  • Utilize as opções para Resetar Senha e Sair do sistema.

Se precisar de mais assistência, entre em contato com o suporte.
