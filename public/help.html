Como Usar o Guara-system

Bem-vindo ao Guara-system! Este webapp foi desenvolvido para gerenciar Licitações e POs de forma rápida e eficiente. Siga as instruções abaixo para tirar o máximo proveito do sistema.

1. Acesso ao Sistema

Ao acessar, você será solicitado a inserir a senha. Use a senha fornecida para continuar.

2. Gerenciamento de Licitações

  • Adicionar Nova Licitação: Clique em "Adicionar nova licitação" no menu flutuante. Preencha os campos obrigatórios, incluindo o CMM (Consumo Mensal Médio) e o Saldo (Disp. p/ lib). A Autonomia será calculada automaticamente dividindo o CMM pelo Saldo.
  • Editar/Excluir: Utilize os ícones de lápis e lixeira em cada cartão para editar ou excluir uma licitação.
  • Exportar Dados: Use o botão "Exportar" para baixar os dados em formato Excel (.xls).
  • Importar Estoque: Clique no botão "Importar Estoque" para carregar uma planilha CSV. O arquivo deve conter as colunas NOME_ITEM e QTDE_DISPONIVEL, que serão usadas para atualizar o saldo (Disp. p/ lib).

3. Gerenciamento de POs

  • Adicionar Novo PO: Clique em "Adicionar novo PO" e preencha o formulário. Selecione a licitação relacionada.
  • Converter PO para OC: No cartão do PO, clique no botão para converter o PO em OC e insira o número da OC quando solicitado.
  • Editar/Excluir/Arquivar: Utilize os ícones correspondentes para editar, excluir ou arquivar POs.
  • Exportar Dados: Clique no botão "Exportar" para baixar os dados de POs em formato Excel (.xls).

4. Pesquisa e Filtros

Use as barras de pesquisa para filtrar por nome e a categoria (Todos, Frigorificados ou Secos) para refinar os resultados.

5. Comentários e Verificação de OCs

  • Comentários: Clique no ícone de chat em cada licitação para visualizar e adicionar comentários.
  • Ver OCs: Clique em "Verificar OCs" para visualizar os OCs associados à licitação.

Se precisar de mais assistência, entre em contato com o suporte.
